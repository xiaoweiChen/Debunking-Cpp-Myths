第3章

—— 惟弃众方可

C++ 起源于“带有对象的 C”，这一历史背景使许多开发者至今仍将它视作一门面向对象编程（OOP）语言。然而，在本章中我们将看到，C++ 实际上支持多种编程范式，你完全可以将它描述为一种“一体多面”的通用编程语言。

我们将深入探讨 C++ 所支持的几种主要编程风格，包括：结构化编程、面向对象编程（OOP）、函数式编程 以及 元编程，并结合其在强类型系统与某些看似“可选类型”风格写法上的表现进行讨论。

在本章中，我们将涵盖以下核心内容：

  • C++ 的多范式特性
  • C++ 中的函数式编程
  • 元编程
  • 强类型的极致使用
  • 忽略类型
