7.1. 环境要求

本章代码可在GitHub获取：https://github.com/PacktPublishing/Debunking-CPP-Myths(位于ch7文件夹)代码使用g++和C++20标准通过Makefile编译。关于Actor Model的示例使用了C++ Actor Framework(CAF)(https://actor-framework.org/)，因此需要先安装该框架。在Ubuntu系统上，可通过运行以下命令安装：apt install libcaf-dev。示例中使用的CAF版本是Ubuntu官方稳定版库：0.17版本。
